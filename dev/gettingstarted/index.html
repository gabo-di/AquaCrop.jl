Install

To install the package you can do

using Pkg
Pkg.add(url="https://github.com/gabo-di/AquaCrop.jl")

Basic Run

AquaCrop.jl is based on FAO's AquaCrop, to mantain compatibility we allow to upload the same configurations files that AquaCrop v7.2 allows, this can be done using the NormalFileRun function

using AquaCrop
runtype = NormalFileRun()

now that we have selected what kind of configuration files we will use, we give the address of the files, for this example we use the files in the test directory

parentdir = AquaCrop.test_dir #".../AquaCrop.jl/test/testcase"
endswith(parentdir, "/AquaCrop.jl/test/testcase")

now we are ready to use the basic_run function

outputs = basic_run(; runtype=runtype, parentdir=parentdir);


isequal(size(outputs[:dayout]), (892, 89))

# output
true

Using TOML files

If you prefer to use TOML and csv files for configuring the input, then use the function TomlFileRun like in the following example

runtype = TomlFileRun();
parentdir = AquaCrop.test_toml_dir  #".../AquaCrop.jl/test/testcase/TOML_FILES"

outputs = basic_run(; runtype=runtype, parentdir=parentdir);

isequal(size(outputs[:dayout]), (892, 89))

# output

true
